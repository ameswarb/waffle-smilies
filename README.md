# waffle-smilies

somethingawful smiley aggregator

* after cloning the repo, run `npm install` and `bower install`. after that, you may issue the following commands:
* `gulp update` pull down smiley metadata from the somethingawful forums. this will be cached in `./src/smilies.json`
* `gulp download` will download any smilies you don't already possess
* `gulp serve` will generate a single page application using the aforementioned metadata, harp, angularjs, and zurb foundation for apps
